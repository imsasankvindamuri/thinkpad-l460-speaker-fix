# thinkpad-l460-speaker-fix
A workaround for a weird Linux audio bug on my Lenovo ThinkPad L460 where
the speaker starts muted after boot or resume.

Made with AI assistance.

## Install

```bash
sudo ./install.sh
```

## Uninstall

```bash
sudo ./install.sh --uninstall
```
